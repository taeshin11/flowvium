→ 요청 시점 UTC 23:10 = KST 08:10 → 장 시작 50분 전 → 빈 응답.

**수정 (src/app/api/korea-flow/route.ts)**:
```typescript
// 최대 7거래일 역방향 스캔, 첫 non-empty 반환
for (let daysAgo = 0; daysAgo <= 7; daysAgo++) {
  const trdDd = kstDateStr(daysAgo);
  const entries = await fetchKrxFlowForDate(market, trdDd);
  if (entries.length > 0) return { entries, trdDd };
}
```

- `tradingDay` payload 필드가 이제 실제 데이터의 거래일 반영 (이전엔 항상 오늘 KST).
