### 발견된 문제
1. market-heatmap v12: KR/TW/IN/CN 요청 시 Stooq pass 불필요하게 실행
   - Stooq는 KR(.kr), TW(.tw), IN(.in), CN(.hk) 모두 N/D 반환 (iter36에서 확인)
   - 80개 ticker 배치 → 2-3 req × 1-2s each = 최대 6s 낭비
   - Pass 1 결과: 0/80 matched (전부 Yahoo fallback으로 넘어감)
   - 해결: STOOQ_SKIP = Set(['KR','TW','IN','CN']) → 해당 국가 Pass 1 건너뜀

2. InsiderPage 탭 순서 불합리:
   - 기존: insider → ownership → nport → blocks → options → korea
   - options/blocks는 API 키 없어서 lock screen만 표시 (중간에 배치됨)
   - korea 탭이 맨 마지막 → 잘 안 보임
   - 해결: insider → korea → ownership → nport → options → blocks

3. Korea 탭 fallback 배너 스타일:
   - ⚠ amber 경고 색상 → 사용자에게 "오류"처럼 보임
   - 실제로는 정상 동작 (KRX 차단은 알려진 제약, 가격 데이터는 정상)
   - 해결: neutral 스타일 (white/5 배경, secondary 텍스트)
