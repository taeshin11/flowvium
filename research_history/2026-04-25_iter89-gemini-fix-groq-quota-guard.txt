#### [2] Groq TPD guard 추가 (ai-providers.ts)
```typescript
let groqTpdExhaustedUntil = 0; // epoch ms

// 8b도 TPD 429이면 다음 UTC midnight까지 Groq skip
if (fallback.status === 429 && fallback.tpdExhausted) {
    const nextMidnight = new Date(Date.UTC(...));
    groqTpdExhaustedUntil = nextMidnight.getTime();
}
// 이후 호출은 Groq skip → 즉시 Gemini
if (groqTpdExhaustedUntil > Date.now()) {
    diag?.push({ ... 'both_tpd_exhausted — skipped' ...});
    return null;
}
```
